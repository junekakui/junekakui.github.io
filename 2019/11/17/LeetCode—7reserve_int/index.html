LeetCode—7reserve_int

Nov 17, 2019

题目概述

第一次思路

  1. 判断整数的正负，用k标记，并且将负数转化为正数
  2. 为了完成溢出返回0，用long数据类型判断整数是否在范围内，2^31-1=2147483647,-2^31=-2147483648

Code

方法总结

  1. 简洁的写法
  2. 官方解答

x是正数时，

如果溢出，则有rev > INT_MAX/10

所以：

当rev > INT_MAX/10时，rev⋅10+pop一定溢出

当rev == INT_MAX/10时，由于2^31-1=2147483647,-2^31=-2147483648，当pop>7时，一定溢出

负数同理

  1. long转化为int判断是否溢出

一个负数除以10得到的除数是负数，所以不需要另外判断x的正负

将y强制转化为int类型，如果溢出，则转化后的数和原来的不相等，可以用来判断是否溢出
